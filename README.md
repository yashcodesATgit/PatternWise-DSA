# PatternWise-DSA (C++) 

A pattern-wise collection of DSA problems (LeetCode / NeetCode) solved in C++.
Each solution is self-contained with full `#include` setup, so it runs directly on any online judge or local compiler — no extra setup needed.

## Structure

Problems are organized by pattern, not by topic. Each pattern has its own folder, and each file inside is one solved problem.

```
PatternWise-DSA/
├── TwoPointer/
│   ├── ContainerWithMostWater.cpp
│   ├── MoveZeroes.cpp
│   └── ...
├── SlidingWindow/
├── PrefixSum/
├── BinarySearch/
├── Stack/
├── Recursion/
├── LinkedList/
├── Tree/
├── Graph/
├── Heap/
├── Backtracking/
├── Greedy/
├── DynamicProgramming/
└── ...
```

## ▶️ How to Run

Each file is standalone — copy any `.cpp` file and run it directly:

```bash
g++ filename.cpp -o solution
./solution
```

Or paste the file content into any online compiler / judge (LeetCode, GFG, OnlineGDB, etc.) — no missing imports, works out of the box.

## 🏷️ Naming Convention

Each file is named after the problem it solves — e.g. `ContainerWithMostWater.cpp`, `MoveZeroes.cpp` — so finding a specific problem inside a pattern folder is quick and obvious.
